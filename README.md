# Stock Market Data Analysis

End-to-end Python data analysis project on AAPL, GOOGL, MSFT and AMZN (2023–2024)

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, yfinance, Power BI

## What This Project Covers
1. Live data ingestion from Yahoo Finance using yfinance
2. Data cleaning and validation
3. Total return and daily returns analysis
4. Volatility and Sharpe ratio calculation
5. Correlation heatmap across all 4 stocks
6. Risk vs return scatter plot
7. 50-day and 200-day moving averages
8. Trading volume analysis
9. Interactive Power BI dashboard

## How To Run
pip install -r requirements.txt

Then open Stock_Analysis_Project_Sai_Suraj_D_.ipynb and click Run All

- Best performer: MSFT with highest total return
